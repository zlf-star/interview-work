# 需求目标
实现一个根据个人简历，AI模拟面试的场景问答助手。
## 功能概述
1. 通过本地选择文件，前端读取文件后进行数据源的输入，可读取pdf、word文档
2. 左侧展示为简历，右侧为智能问答助手对话框，由AI发起提问，用户进行回答后，AI进行分析和回答打分后进行输出，然后进行提问
3. 页面底部有一个总结按钮，可以通过ai总结目前候选人掌握的点和不足之处
4. 左侧有一个更换简历的按钮，更换之前询问用户是否结束以及是否需要总结
## 技术约束
1. 这是一个纯前端业务，不涉及后端业务
2. 使用react+TS+vite+openai接入模型
3. i集成使用deepseek，ai输出使用sse技术，流式展示问答效果
